# weather-website

**Features**
- Get current and weekly forecast.
- Search cities worldwide.
- Switch between imperial and metric systems.
- Responsive design.

**Used Technologies**

- HTML5
- ES6
- SASS
- webpack
- webpack-cli
- VScode

**Third party code**
- date-fns
- google maps
- Font Awsome
